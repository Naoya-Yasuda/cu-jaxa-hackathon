# 先行研究調査: 衛星データによる空き家判定

> 調査日: 2026-02-11

---

## 調査目的

衛星データ（リモートセンシング）を活用した空き家判定・検出に関して、既存の商用サービス、政府プロジェクト、学術研究を網羅的に調査し、本プロジェクトの差別化ポイントを明確にする。

---

## 1. 結論サマリ

衛星データによる空き家判定は**既にサービス化・研究が進んでいる分野**。ただし「衛星データ単独」での個別空き家判定は精度に限界があり、**行政データやストリートビュー等との組み合わせ**が主流。

| 分類 | 成熟度 | 代表例 |
|---|---|---|
| 衛星+行政データの商用サービス（日本） | ★★★★★ 実用化済 | Penetrator/WHERE、パスコ |
| リモセン+AIでエリア単位の推定（日本） | ★★★★ 実証済 | AKIDAS（空き家活用）、サグリ |
| 国の空き家推定システム | ★★★★ 実証中 | 国交省 LINKS SOMA |
| 夜間光データで空室率推定（海外） | ★★★ 研究蓄積多 | 中国の大学群（VIIRS/Luojia-1） |
| 熱赤外衛星で建物使用状態判定（海外） | ★★★ 衛星構築中 | Satellite Vu（英国） |
| VHR衛星画像のみで個別空き家検出 | ★★ 精度不十分 | Zou & Wang（生産者精度39.1%） |

## 7. 技術的課題と知見

### 7-1. 衛星データ単独の限界

1. **空間解像度の壁**: 現行の無料衛星データ（VIIRS ~750m、JAXA NDVI ~5km、Sentinel-2 10m）では個別建物の居住判定は困難
2. **間接指標への依存**: 植生変化（NDVI）、夜間光、建物劣化状態など間接指標の組み合わせが必須
3. **上方視点のみ**: 衛星は上からしか見えない。建物正面の状態（窓の破損、外壁劣化）は見えない → ストリートビュー統合が有効

### 7-2. 精度向上のカギ

- **最も精度が高いアプローチ**: 行政データ（水道使用量、住基台帳、登記情報）+ 機械学習 → **精度95.4%**
- **衛星データの役割**: 行政データが利用できない場合の代替手段、または行政データと組み合わせる補助的な特徴量
- **無料データのみでの最善策**: Sentinel-2 NDVI（10m解像度）の経年変化 + 公開統計データ（国勢調査、住宅統計）

### 7-3. 今後のトレンド

1. **高解像度熱赤外衛星の普及**: Satellite Vuのコンステレーション完成で建物レベルの判定が可能に
2. **マルチモーダルデータ融合**: 衛星光学+夜間光+SAR+ストリートビュー+行政データの統合
3. **LINKS SOMAの全国展開**: オープンソースで自治体に無料提供 → 空き家推定の標準インフラ化
4. **ディープラーニングの高度化**: 屋根劣化検出、植生管理状態判定、建物経年変化分析の精度向上

## 8. 差別化のポイント（本プロジェクトの立ち位置）

### 既存サービスの共通特徴

| 特徴 | 詳細 |
|---|---|
| **ターゲット** | 不動産業者（BtoB）または自治体（BtoG） |
| **データ** | 高価な商用衛星データ、または非公開の行政データ（水道・住基台帳）を使用 |
| **目的** | 空き家の「発見・特定」に特化 |
| **UI** | プロフェッショナル向けの複雑なシステム |
| **規模** | 全国カバーの大規模システム |

### 差別化の方向性

| 既存サービスの特徴 | 本プロジェクトの差別化 |
|---|---|
| 高価な商用衛星データや非公開行政データを使用 | **無料のJAXA Earth API + 公開オープンデータのみ**で構築 |
| BtoB/BtoG向け（不動産業者・自治体） | **市民・住民向け（BtoC）**の空き家情報サービス |
| 空き家の「発見」に特化 | 空き家の「活用」「移住」「防災」など**出口の設計**まで含む |
| プロフェッショナル向けの複雑なUI | **誰でも使えるシンプルなWebアプリ** |
| 全国カバーの大規模システム | **特定の地方自治体に特化**したプロトタイプ |
| サイバー大学生との接点なし | **サイバー大学生の特性**（通信制・社会人・地方在住）を活かした設計 |
