## 1. JAXA APIで取得できるデータ一覧

### A. JAXA Earth API（data.earth.jaxa.jp）★最も開発者フレンドリー

| 項目 | 詳細 |
|---|---|
| **URL** | https://data.earth.jaxa.jp |
| **認証** | 不要（登録もAPIキーも不要） |
| **データ形式** | COG (Cloud Optimized GeoTIFF) + STAC準拠 |
| **SDK** | Python API (v0.1.5), JavaScript API (v1.2.3) |
| **AI連携** | MCP（生成AI連携）対応 |
| **データセット数** | 90件以上 |
| **料金** | 無料（商用利用も条件付きで可能） |

| カテゴリ | データセット | 識別子例 | 空間分解能 |

| 降水量 | GSMaP 衛星全球降水マップ | - | - |
| 海面水温 | GCOM-C SST | - | 約250m |
| 海面水温 | GCOM-W SST（雲貫通型） | - | 低解像度 |
| クロロフィルa | GCOM-C 植物プランクトン分布 | - | 約250m |
| 地表面温度 | LST（昼夜別） | - | 約5km |
| 植生指数 | NDVI | - | 約5km |
| 土壌水分量 | GCOM-W（昼夜別、月/日平均） | - | - |
| 海氷 | 北極・南極海氷密接度（日次） | - | - |
| 温室効果ガス | CO2等濃度（いぶき2号/GOSAT-2） | - | - |

## 2. 空き家判定アプリの評価

### コンセプト
夜間光などの衛星データで空き家かどうかを自動判定するアプリ。「地方 x 孤立」というサイバー大学生の特性にフィットさせる。

### 技術的実現性の評価

#### 夜間光データで空き家を判定できるか？

| アプローチ | 実現性 | 理由 |
|---|---|---|
| **1軒単位の空き家判定** | ❌ 不可 | 衛星の夜間光データ（NASA/NOAA VIIRS等）の解像度は500m〜1km四方。1ピクセルに数百〜数千世帯が含まれるため、個別の家の明かりは識別不可能 |
| **町丁目・エリア単位の「空き家率が高そうなエリア」推定** | ⭕ 可能 | 夜間光の経年変化（光が減少しているエリア）+ NDVI（植生増加＝管理放棄の兆候）の組み合わせで統計的に推定可能 |
| **AI学習の特徴量の1つとして** | ⭕ 有望 | 衛星データを他のオープンデータ（住民基本台帳、固定資産税、水道使用量等）と組み合わせれば、空き家確率を算出するモデルの有力な入力特徴量になる |

#### JAXA データで空き家推定に使えるもの

| データ | 活用方法 | 有効性 |
|---|---|---|
| **NDVI（植生指数）** | 管理放棄された土地は植生（雑草・樹木）が急増する。経年比較で「急に緑が増えた宅地」を検出 | ★★★★（最も有望） |
| **AW3D30（数値表層モデル）** | 建物密度が低いエリアの特定、地形的に孤立した集落の把握 | ★★★ |
| **LST（地表面温度）** | 人が住んでいるエリアと無人エリアの温度差（ヒートアイランド効果の逆）。ただし解像度5kmのため粗い | ★★ |
| **GSMaP（降水量）** | 災害リスクの高い空き家エリアの特定（豪雨常襲地帯の空き家＝危険空き家） | ★★★ |
| **ALOS-4 SAR画像** | 合成開口レーダーで建物の物理的な状態（倒壊・劣化）を検出できる可能性。ただし高度な解析が必要 | ★★（研究段階） |

#### 夜間光データの入手について（補足）
JAXAのAPIでは夜間光（ナイトタイムライト）データは直接提供されていない。夜間光データを使う場合は以下の外部ソースが必要：

| データソース | 解像度 | 提供元 |
|---|---|---|
| VIIRS DNB (Day/Night Band) | 約500m | NASA/NOAA（無料） |
| DMSP-OLS | 約1km | NOAA（無料、アーカイブ） |
| Luojia-1 | 約130m | 武漢大学（限定的） |

### サイバー大学生との親和性評価

#### 強み
- **ビジネス性**: 空き家問題は日本の社会課題（全国約849万戸、2023年時点）。自治体・不動産業者にニーズがある
- **IT/データサイエンス実践**: 衛星データ解析 + 機械学習という技術的チャレンジがある
- **地方課題**: 通信制で地方在住の学生が多い → 自分の地域の課題解決に直結

#### 弱み・要検討事項
- **「サイバー大学生がユーザーになる」動機**: 空き家データの主要ユーザーは自治体・不動産業者・投資家。学生自身が空き家情報を必要とするシーンの設計が必要
- **衛星データだけでは精度不足**: 個別物件レベルの判定には、自治体のオープンデータ（水道使用量、住民登録等）との組み合わせが不可欠
- **既存競合**: 空き家バンク（自治体運営）、LIFULL HOME'S空き家バンク等が既に存在

### 方向性の候補

#### A案: 「空き家×移住支援」アプリ
サイバー大学生（特に社会人学生）が地方移住を検討する際に、衛星データで地域の環境を評価しながら空き家物件を探せるアプリ。通信制なので住む場所を選ばない特性を活かす。

#### B案: 「空き家リスク可視化」ダッシュボード
自治体向けに、衛星データ（NDVI経年変化 + 降水量 + 地形）から空き家リスクの高いエリアをヒートマップ表示。サイバー大学生がチームで開発し、地元自治体に提案するプロジェクト型。

#### C案: 「空き家×防災」アプリ
空き家 + 災害リスク（GSMaP降水量、地形データ）を掛け合わせて「危険空き家」を自動検出。防災観点から社会的意義が高く、自治体への提案もしやすい。

## 3. 既存サービス・研究の調査

> 調査日: 2026-02-11

### 結論サマリ

衛星データによる空き家判定は**既にサービス化・研究が進んでいる分野**。ただし「衛星データ単独」での個別空き家判定は精度に限界があり、**行政データやストリートビュー等との組み合わせ**が主流。

| 分類 | 成熟度 | 代表例 |
|---|---|---|
| 衛星+行政データの商用サービス（日本） | ★★★★★ 実用化済 | Penetrator/WHERE、パスコ |
| リモセン+AIでエリア単位の推定（日本） | ★★★★ 実証済 | AKIDAS（空き家活用）、サグリ |
| 国の空き家推定システム | ★★★★ 実証中 | 国交省 LINKS SOMA |
| 夜間光データで空室率推定（海外） | ★★★ 研究蓄積多 | 中国の大学群（VIIRS/Luojia-1） |
| 熱赤外衛星で建物使用状態判定（海外） | ★★★ 衛星構築中 | Satellite Vu（英国） |
| VHR衛星画像のみで個別空き家検出 | ★★ 精度不十分 | Zou & Wang（生産者精度39.1%） |

### 3-1. 日本国内の商用サービス

#### (1) 株式会社Penetrator（旧WHERE）— 最も先行

| 項目 | 詳細 |
|---|---|
| **サービス名** | WHERE（不動産取引支援SaaS） |
| **設立** | 2022年2月（JAXA認定ベンチャー） |
| **使用データ** | 光学衛星画像 + 水道開栓/閉栓データ + 登記情報 + 地理空間データ |
| **手法** | JAXAの月面クレーター分析用AIを転用。衛星画像から屋根の錆・変色・構造劣化を検出。水道データと組み合わせて空き家を特定 |
| **精度** | 相模原市津久井地区の実証で、閉栓住宅1,200軒→579軒に絞り込み。自治体調査との一致率67%。屋根劣化・庭の管理放棄が進んだ住宅は**100%正しく検出** |
| **実績** | 月間売上1億円突破（2024年9月正式版リリースから9か月）。約50社の法人顧客。2025年4月にシリーズAで5.5億円調達 |
| **現状** | **サービス継続中・急成長中** |

#### (2) 空き家活用株式会社（AKIDAS）

| 項目 | 詳細 |
|---|---|
| **サービス名** | AKIDAS（日本最大級の空き家データベース） |
| **使用データ** | 熱赤外線衛星画像 + 可視光衛星画像 + サグリ社ACTABA（耕作放棄地検出AI） |
| **手法** | リモセンデータをAIで解析し、90mメッシュ単位で空き家密集エリアを推定 |
| **精度** | 80%以上のメッシュで空き家密集エリアを特定可能（目標値） |
| **実績** | 2020年 東京都「民間空き家対策東京モデル支援事業」採択。世田谷区・墨田区で実証 |
| **現状** | **AKIDASサービス継続中** |

#### (3) 株式会社パスコ

| 項目 | 詳細 |
|---|---|
| **使用データ** | 光学衛星画像（超解像処理適用） |
| **手法** | 異なる時期の衛星画像をAIで比較し、建物の異動（新築・増築・解体）を検出。固定資産課税業務の効率化 |
| **精度** | 福井県坂井市で約60,000棟を対象に実証 |
| **現状** | **2024-2025年実証。事業化済** |

#### (4) 株式会社スペースシフト

| 項目 | 詳細 |
|---|---|
| **使用データ** | SAR衛星データ（Sentinel-1等）+ 3D都市モデル（PLATEAU） |
| **手法** | SAR画像のAI解析で新規建物検出・都市開発モニタリング。InSAR技術でミリメートル単位の地表変位検出 |

### 3-5. 技術別の空き家検出精度まとめ

| 技術/データ | 空間解像度 | 空き家検出精度 | 適用レベル |

### 3-6. 差別化のポイント（既存サービスとの違いをどう出すか）

既存サービスは主に**不動産業者・自治体向け（BtoB/BtoG）**。サイバー大学生のプロジェクトとして差別化するなら：

| 既存サービスの特徴 | 差別化の方向性 |
|---|---|
| 高価な商用衛星データや非公開行政データを使用 | **無料のJAXA Earth API + 公開オープンデータのみ**で構築 |
| BtoB/BtoG向け（不動産業者・自治体） | **市民・住民向け（BtoC）**の空き家情報サービス |
| 空き家の「発見」に特化 | 空き家の「活用」「移住」「防災」など**出口の設計**まで含む |
| プロフェッショナル向けの複雑なUI | **誰でも使えるシンプルなWebアプリ** |
| 全国カバーの大規模システム | **特定の地方自治体に特化**したプロトタイプ |
